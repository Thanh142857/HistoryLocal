# History Local API

## Technologies

- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Swagger

## Features

- Create post
- Update post
- Delete post
- Search by location
- Search by event date
- Pagination
- Validation
- Global Exception Handling

## Swagger

http://localhost:8080/swagger-ui/index.html

## Run Project

1. Clone repository
2. Configure database
3. Run Spring Boot application
